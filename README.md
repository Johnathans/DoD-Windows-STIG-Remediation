# DoD-Windows-STIG-Remediation

This repository demonstrates remediation of Windows 10 STIG findings using PowerShell.

## Lab 1 – Disable SMBv1

### MITRE ATT&CK Mapping
- T1021.002 – SMB/Windows Admin Shares
- T1210 – Exploitation of Remote Services

### NIST 800-53 Mapping
- CM-7 – Least Functionality
- SI-2 – Flaw Remediation

### Objective
Disable the deprecated SMBv1 protocol to reduce attack surface.
